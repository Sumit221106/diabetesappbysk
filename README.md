# Diabetes Prediction App 🩺

A Machine Learning web application built using Python and Flask that predicts whether a person is likely to have diabetes based on medical input data.

## 🚀 Features
- User-friendly web interface
- Predicts diabetes using trained ML model
- Built with Flask
- Uses Pickle model file (`diab.pkl`)

## 🛠️ Technologies Used
- Python
- Flask
- HTML/CSS
- Machine Learning
- Pickle

## 📂 Project Structure
```bash
templates/
app.py
diab.pkl
README.md
```

## ▶️ How to Run

1. Install required libraries
```bash
pip install flask numpy pandas scikit-learn
```

2. Run the application
```bash
python app.py
```

3. Open browser and visit
```bash
http://127.0.0.1:5000
```
